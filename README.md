# 峡谷消消乐

这是一个完整可玩的三消游戏原型：动态峡谷背景、艺术字标题、英雄头像棋子、连击播报、技能棋子、羁绊彩蛋、结算页和语音触发管线都已经接好。

## 怎么玩

直接打开 `index.html`。交换相邻头像，凑齐三个以上就会消除；四连和五连会生成技能头像，连锁越长，播报越夸张。

## 关于《王者荣耀》素材

英雄原画、英雄名称、台词原声、音效和相关 UI 资产都属于受版权或商标保护的内容。商业上线或公开传播前，需要腾讯/王者荣耀官方授权。

当前版本使用原创占位英雄和合成音效。拿到授权素材后，可以按 `asset-template.json` 的格式接入：

- 图片放到 `assets/heroes/`
- 语音放到 `assets/voices/`
- 在 `game.js` 的 `heroes` 数组中替换 `name`、`quote`、`voice`
- 如果要显示真实头像，再把 `.tile` 的背景逻辑改成读取授权图片
