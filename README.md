# 贴贴消消乐

一个参考视频风格制作的手机端消消乐网页原型：奶油纸张背景、手帐贴纸动物、5x5 清盘棋盘、Ready/Go、Good/Perfect、道具和轻快合成音效。

## 怎么玩

直接打开 `index.html`，或发布到 GitHub Pages 后在手机浏览器访问。

- 点击上下左右相邻的同款贴纸组，2 个以上即可消除。
- 消除后格子留空，不自动掉落补位。
- 清空整张棋盘后进入 `win` 结算动画。
- 底部三个道具分别是提示、炸弹和洗牌。
- 第一次点击棋盘或道具后会启动背景音乐和音效。

## 设计参考

当前版本重点还原视频里的体验方向：

- 竖屏手机优先。
- 5 列棋盘、圆角贴纸格、手绘描边。
- 淡奶油背景和底部浅黄色波浪地面。
- 轻弹点击反馈、星星粒子、泡泡字评价。
- 背景音乐与消除音效使用 Web Audio 合成，避免外部授权素材依赖。

## 发布

这是纯静态项目，不需要构建步骤。GitHub Pages 可以直接选择仓库根目录发布。
