# 萌宠消消乐

一个手机端可玩的标准 Match-3 消消乐网页游戏原型。美术方向参考奶油手帐贴纸风，但玩法回到主流消消乐：交换相邻棋子、形成横向或纵向三连、消除、下落补位、自动连锁。

## 怎么玩

直接打开 `index.html`，或访问 GitHub Pages 发布地址。

- 点击两枚相邻萌宠，或在手机上滑动一枚萌宠，与相邻位置交换。
- 交换后必须形成横向或纵向 3 个以上同款萌宠，才会消除。
- 无效交换会自动换回，不扣步数。
- 消除后上方萌宠下落，顶部补充新萌宠，并继续检测连锁消除。
- 4 连会生成横向或纵向特效萌宠。
- 5 连会生成彩虹特效萌宠。
- T/L 形消除会生成爆炸特效萌宠。
- 每关有步数和收集目标，目标完成即可胜利。

## 道具

- 放大镜：提示一个可交换位置。
- 小锤：直接敲掉一枚萌宠并触发下落补位。
- 洗牌：重排当前棋盘，保证还有可走步。

## 设计方向

- 竖屏手机优先。
- 奶油纸张背景、浅黄色底部波浪、手绘描边。
- 棋子是明确可识别的萌宠头像：猫、狗、兔、熊、鸭、熊猫。
- Ready/Go、Good/Great/Perfect 反馈。
- 背景音乐和音效用 Web Audio 合成，避免外部素材授权依赖。

## 发布

这是纯静态项目，不需要构建步骤。GitHub Pages 可以直接选择仓库根目录发布。
